# Malaria Cell Image Classification

## Overview
This project focuses on classifying malaria-infected and uninfected cell images using machine learning and deep learning techniques. The goal is to support early and accurate malaria diagnosis using automated image analysis.

## Dataset
- Dataset: Malaria Cell Images Dataset
- Type: Binary classification (Parasitized vs Uninfected)
- Image-based dataset

## Methodology
- Data preprocessing and cleaning
- Image resizing and normalization
- Train-validation-test split
- Model development:
  - Custom CNN
  - Transfer Learning (EfficientNet / ResNet)
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-score

## Results
- Best performing model: ResNet50
- Accuracy: 96%

## Tools & Technologies
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib / Seaborn

## Key Takeaways
- Deep learning is highly effective for medical image classification
- Transfer learning improves performance and reduces training time
- Proper preprocessing is critical for model accuracy
